# linux
## 1、	小命令
(1) unmae -m 显示电脑的cpu架构  
(2) uname -a 显示内核版本  
(3)	 在proc目录下存储了 几个文件  
   a)	 cpuinfo 存储的cpu的详细信息  
   b)	 version  存储的内核的版本  
(4)	date  显示当前系统的日期  
(5)	cal 显示当月日历  
(6)	cal 年份  显示这一年整年的日历
